# Pillgrim

A FOSS medication reminder app that runs entirely on the device and contains no ads or analytics.

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

</div>

## Key Features

### Flexible Scheduling
Set schedules for daily, weekly (specific days), every N days, twice a month, specific dates, or "as needed" (PRN). Add multiple dose times per day with custom amounts, dietary instructions, stock tracking with low-stock alerts, and optional photos for identification.

### Smart Notifications
Toggle reminders per medication (not a single global switch). Hide medication names on the lock screen for privacy. Notification actions include marking as taken, skipping, or snoozing.

### Dose Logging & Adherence
Doses are automatically tracked as pending, taken, skipped, or missed. View a weekly calendar on the main screen or a full month view with adherence color dots (green/yellow/red). Filter and export your complete log history as a CSV or PDF.

### Medication Management
Pause medications temporarily instead of deleting them—great for finished courses or holds. Deleting a medication also removes its history, with a short undo window available.

### Custom Appearance
Supports light/dark/system themes, 12/24-hour time, configurable first day of the week, and independent font scaling (four sizes). The interface is translated into 13 languages and adapts to your device's language setting.

### 100% Private
All data stays locally on your device. No servers, no accounts, no analytics—nothing leaves your phone. Logs are automatically pruned after 5 years to save space.

## Getting Started
```
bash

git clone https://github.com/leidorf/pillgrim.git
cd pillgrim
npm install
npx expo start
```

Then open the project in Expo Go, an iOS Simulator, or an Android emulator.
